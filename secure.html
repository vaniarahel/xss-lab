Teknik Keamanan yang Digunakan

  • textContent - Menampilkan input sebagai teks, bukan HTML
  • createElement - Membuat elemen secara programatis
  • Input Validation - Membatasi panjang dan karakter
  • CSP Header - Membatasi sumber script

Perbandingan Kode

VULNERABLE

// JANGAN LAKUKAN INI!
container.innerHTML = `
  <div class="comment">
    <strong>${username}</strong>
    <p>${comment}</p>
  </div>
`;

SECURE

// Gunakan textContent & createElement
const card = document.createElement('div');
card.className = 'comment';

const name = document.createElement('strong');
name.textContent = username; // SAFE!

const text = document.createElement('p');
text.textContent = comment; // SAFE!

card.appendChild(name);
card.appendChild(text);
container.appendChild(card);
